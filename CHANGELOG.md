# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

## [1.0.0] - 2026-03-11

### Added
- Homepage with app card navigation
- Vibe Planner app with Supabase authentication
- Multi-user support with Row Level Security
- Board view — Kanban-style drag & drop across 5 statuses
- Timeline view — weekly calendar with past/future navigation
- List view — compact sortable list with quick status cycling
- Built-in per-task time tracking
- Category tagging (Work, Personal, Creative, Learning, Health)
- Mobile-responsive design
- Cross-device data sync via Supabase

## [0.1.0] - 2026-03-11

### Added
- Initial Vibe Planner with localStorage persistence
- Export/Import functionality for JSON backups
- Single HTML file, zero dependencies
